# IA_Reconnaissance
C'est une plateforme unifiée de reconnaissance faciale sous Streamlit. Elle intègre de façon native la gestion de profils, le recadrage automatique de visages par MTCNN, l'apprentissage sur-mesure d'un modèle MobileNetV2 (Deep Learning) et l'analyse via photo fixe ou flux Webcam Live direct sans plantage de processus.
